# poshukBot
Telegram-бот для пошуку нових оголошень OLX по нерухомості в Житомирській області.

## Запуск

1. Встановити залежності:

```powershell
pip install -r requirements.txt
```

2. Перевірити, що у файлі `.env` є токен:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

3. Запустити бота:

```powershell
python main.py
```

4. У Telegram написати боту `/start`.

## Файли

- `config.json` - категорії пошуку та фільтри.
- `seen.json` - уже побачені оголошення, щоб не надсилати дублікати.
- `users.json` - chat id користувачів, яким бот надсилає повідомлення.
- `data/ads/` - збережені знайдені оголошення по населених пунктах.

## Як користуватись

- Після `/start` бот показує одну нижню клавіатуру біля поля вводу.
- Inline-кнопки під повідомленнями не використовуються.
- `Перевірити зараз` запускає ручну перевірку.
- `Категорії` вмикає або вимикає напрямки пошуку.
- `Містечка` показує населені пункти, куди бот уже зберіг знайдені й відфільтровані оголошення.
- У розділі `Містечка` натисніть населений пункт, щоб побачити тільки його оголошення.
- `Житомир` завжди є в розділі `Містечка`; бот також розпізнає форми `Житомира` та `Житомирі`.
- Кнопка `⬅️ До містечок` ховає показані оголошення і повертає список містечок.
- Під кожним оголошенням є маленька кнопка `Зберегти в обране`.
- `Обране` показує збережені вами оголошення.
- `Фільтри` налаштовує площу, ціну, ключові слова та стоп-слова.
- Автоматична перевірка запускається щогодини, якщо встановлено `python-telegram-bot[job-queue]`.

Пошук налаштований на OLX-регіон `zht`, тобто всю Житомирську область. Після фільтрації бот визначає населений пункт з тексту оголошення і зберігає його в `data/ads/`.
